# Sample Reactive - Spring Boot application

코틀린 기반 Reactive 프로젝트 입니다. 해당 프로젝트는 Kotlin 의 co-router와 coroutine 을 이용하여 개발되었습니다.
DB는 [PostgreSQL](https://github.com/r2dbc/r2dbc-postgresql) 와 R2DBC를 사용하였습니다.

---
## 스펙
* Java 11
* Spring Boot 2.7.8
* R2DBC
* Webflux + coroutin
* PostgreSQL
* liquibase
* springdoc-openapi
